<br>

- Agentes con herramientas personalizadas (Tool Use)
- Orquestación de workflows con `SequentialAgent` y `ParallelAgent`
- Integración con Vertex AI Gemini como modelo base
- Memoria y estado persistente entre turnos

</details>

<details>
<summary><b>💬 Conversational Agents & CCAI</b></summary>
<br>

- **Dialogflow CX / Conversational Agents:** diseño de flujos conversacionales avanzados, intents, entidades, webhooks, conditional routes
- **Agent Studio:** prototipado rápido y testing de agentes en la plataforma Google CCAI
- **Genesys Cloud CX:** integración de bots de voz y texto con Dialogflow, diseño de contact flows, IVR y routing inteligente
- **CCAI Insights:** análisis de conversaciones, sentiment analysis y topic modeling en llamadas

</details>
